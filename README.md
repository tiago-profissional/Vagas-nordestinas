# Vagas Nordestinas

Vagas Nordestinas is a full-stack job board platform built with React, PHP, and MySQL.  
It helps companies and recruiters connect with candidates, allowing users to post and apply for jobs focused on the Northeast region of Brazil.

Every part of this project was developed to demonstrate practical full-stack development skills, including:

* Build a responsive frontend interface with React  
* Create a complete CRUD system for job posting management  
* Develop a RESTful backend API using PHP  
* Store and manage job and user data with MySQL  
* Implement authentication and user registration/login flows  
* Create a dashboard for managing job posts and platform activity  
* Connect frontend and backend in a real-world full-stack workflow

## Developer Instructions

This section explains how to set up and run the project locally for development.

### Frontend

- Built with React  
- Handles UI, routing, and API communication  

Steps:

1. Navigate to the frontend directory  
2. Install dependencies:  
   npm install  
3. Run the development server:  
   npm run dev  

---

### Backend

- Built with PHP  
- Provides REST API endpoints  
- Connected to a MySQL database  

Steps:

1. Place the backend folder inside XAMPP's htdocs directory  
2. Start Apache and MySQL  
3. Configure database credentials in the config file  
4. Import the SQL database  
5. Access the API via localhost  

---

### Notes

- Ensure XAMPP is running before testing the backend  
- Update API base URL in the frontend if needed
